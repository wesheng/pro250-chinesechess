## 中国象棋
### 概述
编程语言：C#  
IDE : Visual Studio 2017  
.NET：.Net FrameWork 4.6.1

### 功能
功能在右键菜单内
* 单机象棋对战
* 支持空过、支持悔棋一步
* 选定棋子会有闪烁效果，移动到己方棋子会取消所有选定
* 对不可选位置、不可移动位置有鼠标指针提示
